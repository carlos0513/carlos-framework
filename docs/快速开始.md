# Carlos Framework - 快速开始指南

本文档帮助你快速配置和使用 Carlos Framework 的 Claude Code 开发环境。

---

## 📋 前置要求

- ✅ Node.js 和 npm/npx（已安装）
- ✅ JDK 21+（已安装）
- ✅ Maven 3.8+（已安装）
- ✅ pnpm（前端开发需要）
- ⚠️ MySQL 数据库（需配置连接）
- ⚠️ ClickHouse 数据库（可选，用于数据分析）

---

## 🚀 5 分钟快速配置

### 1. 设置环境变量（必需）

**Windows PowerShell：**

```powershell
# GitHub Token（用于代码搜索和 PR 管理）
# 在 https://github.com/settings/tokens 生成，需要 repo 权限
$env:GITHUB_TOKEN = "ghp_your_token_here"

# MySQL 数据库连接（用于业务数据查询）
$env:MYSQL_HOST = "localhost"
$env:MYSQL_PORT = "3306"
$env:MYSQL_USER = "root"
$env:MYSQL_PASSWORD = "your_mysql_password"
$env:MYSQL_DATABASE = "carlos"

# ClickHouse 数据库连接（可选，用于数据分析）
$env:CLICKHOUSE_HOST = "localhost"
$env:CLICKHOUSE_PORT = "8123"
$env:CLICKHOUSE_USER = "default"
$env:CLICKHOUSE_PASSWORD = "your_clickhouse_password"
$env:CLICKHOUSE_DATABASE = "default"
```

**永久设置（推荐）：**

```powershell
# 设置 GitHub Token
[Environment]::SetEnvironmentVariable("GITHUB_TOKEN", "ghp_your_token_here", "User")

# 设置 MySQL 连接
[Environment]::SetEnvironmentVariable("MYSQL_HOST", "localhost", "User")
[Environment]::SetEnvironmentVariable("MYSQL_PORT", "3306", "User")
[Environment]::SetEnvironmentVariable("MYSQL_USER", "root", "User")
[Environment]::SetEnvironmentVariable("MYSQL_PASSWORD", "your_password", "User")
[Environment]::SetEnvironmentVariable("MYSQL_DATABASE", "carlos", "User")

# 设置 ClickHouse 连接（可选）
[Environment]::SetEnvironmentVariable("CLICKHOUSE_HOST", "localhost", "User")
[Environment]::SetEnvironmentVariable("CLICKHOUSE_PORT", "8123", "User")
[Environment]::SetEnvironmentVariable("CLICKHOUSE_USER", "default", "User")
[Environment]::SetEnvironmentVariable("CLICKHOUSE_PASSWORD", "your_password", "User")
[Environment]::SetEnvironmentVariable("CLICKHOUSE_DATABASE", "default", "User")
```

**重启终端**使环境变量生效。

## ⚠️ 常见问题

### Q1: MCP 无法连接数据库

**问题：** 提示 `MYSQL_PASSWORD not set` 或连接失败

**解决方案：**

1. 确认环境变量已设置：`echo $env:MYSQL_PASSWORD`
2. 重启终端使环境变量生效
3. 检查数据库是否启动：`mysql -h localhost -u root -p`
4. 检查防火墙设置

### Q2: Maven 构建失败

**问题：** `mvn clean install` 失败

**解决方案：**

```bash
# 使用构建错误修复 Agent
@carlos-build-resolver mvn clean install 失败了，请帮我诊断

# 或手动排查
mvn clean install -e -X  # 查看详细错误信息
mvn dependency:tree      # 检查依赖冲突
```

### Q3: 前端启动失败

**问题：** `pnpm dev` 失败

**解决方案：**

```bash
# 清理并重新安装依赖
cd carlos-ui
rm -rf node_modules pnpm-lock.yaml
pnpm install

# 检查 Node.js 版本
node --version  # 需要 Node.js 18+
```

### Q4: GitHub Token 权限不足

**问题：** GitHub MCP 提示权限错误

**解决方案：**

1. 访问 https://github.com/settings/tokens
2. 生成新 Token，勾选 `repo` 权限
3. 更新环境变量：`$env:GITHUB_TOKEN = "ghp_new_token"`
